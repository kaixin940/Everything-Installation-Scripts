EverythingToolbar 手动安装指南
=====================================

EverythingToolbar 是一个将 Everything 搜索集成到 Windows 任务栏的工具

前置要求：
- Windows 10 或 Windows 11
- Everything 1.4.1 或更高版本已安装并运行
- .NET Framework 4.7 或更高版本

安装方法：

方法1：从 GitHub 下载（推荐）
1. 访问：https://github.com/srwi/EverythingToolbar/releases
2. 下载最新版本的 EverythingToolbar-Launcher-Setup.exe
3. 运行安装程序
4. 按照安装向导完成安装

方法2：使用包管理器
如果有 winget：
winget install srwi.everythingtoolbar.launcher

如果有 Chocolatey：
choco install everythingtoolbar

安装后配置：

Windows 11 用户：
- 安装后会自动启动设置向导
- 按照向导指示完成配置
- 任务栏会出现搜索图标

Windows 10 用户：
- 右键点击任务栏空白处
- 选择"工具栏" -> "EverythingToolbar"
- 如果第一次没看到，请再次右键任务栏
- 解锁任务栏以调整大小和位置

使用说明：
- 点击任务栏上的搜索框
- 输入要搜索的文件或文件夹名称
- 支持所有 Everything 的搜索语法：
  - filename.ext 搜索特定文件
  - foldername 搜索文件夹
  - *.txt 搜索特定扩展名
  - file:>1mb 按大小搜索
  - path:d:\ 按路径搜索

故障排除：
1. 如果工具栏没有出现：
   - 重启 Windows 资源管理器
   - 确认 Everything 正在运行
   - 检查任务栏 -> 工具栏菜单

2. 如果搜索无结果：
   - 确认 Everything 服务正在运行
   - 检查 EverythingToolbar 设置中的实例名称
   - 等待 Everything 完成文件索引

3. 对于 Everything 1.5a 用户：
   - 在 EverythingToolbar 设置中
   - 高级 -> Everything 实例名称设置为 "1.5a"

卸载：
- 通过 Windows 控制面板卸载
- 或使用 winget uninstall srwi.everythingtoolbar.launcher
